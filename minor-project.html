Minor Project: Digital Alarm Clock with Thermometer

About the Project

A digital clock was built which displayed time and date along with the ambient temperature. It was an 8051 based system which included 16X2 LCD for display, LM35 for temperature sensing, and ADC0804 for interfacing sensor with 8051. It had inputs in order to adjust date and time.

Project Components

  • 8051 microcontroller
  • 16X2 LCD display
  • LM35 temperature sensor
  • ADC0804 ADC converter

Project Features

  • Display of time and date
  • Ambient temperature measurement
  • User inputs for adjusting date and time
